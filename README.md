# Setup

```sh
# 0. clone fork project
git clone https://github.com/Yourname/Codeit_Costory.git

# 1. virtual environment setup
python -m venv VENV

# 2. venv run
# windows
source myvenv/Scripts/activate

# MacOS
source myvenv/bin/activate

# 3. runserver
cd hgg
python manage.py runserver
```
